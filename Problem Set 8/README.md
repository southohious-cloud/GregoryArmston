# 🧩 Problem Set 8

## Overview  
Problem Set 8 introduces object‑oriented programming, emphasizing classes, methods, encapsulation, and clean program structure. These exercises strengthen your ability to model real‑world entities, manage internal state, and design reusable components.

## Files  
1. **`01_seasons.py`**  
   Calculates how many minutes a person has been alive based on their birthdate, reinforcing date parsing, validation, and class‑based organization.

2. **`02_jar.py`**  
   Implements a `Jar` class that tracks cookie capacity and consumption, practicing attributes, methods, error handling, and state management.

3. **`03_shirtificate.py`**  
   Generates a CS50‑style certificate using the `FPDF` library, reinforcing class usage, layout control, and structured PDF creation.
